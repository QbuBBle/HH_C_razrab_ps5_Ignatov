# HHParser

## Описание
Парсер данных с сайта hh.ru. Введя в поиск “C# разработчик”, рассматриваем все регионы. 
Данные которые парсятся: 
* Название вакансии 
* Требуемый опыт работы 
* Заработную плату 
* Регион

Результаты записываются в JSON файл.

## Как использовать?
Запустить файл "parser.py"
